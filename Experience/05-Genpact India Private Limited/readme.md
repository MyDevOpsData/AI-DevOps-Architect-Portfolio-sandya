## Genpact India Private Limited (AI / MLOps / DevOps Project)(2024/March to 2025/April)

### 📌 Overview
This project reflects my experience working on AI/ML pipelines integrated with DevOps practices, focusing on automation, scalability, and production-grade ML deployments.

It combines MLOps, DevOps, and Cloud AI services using GCP and modern tools.

---

## 🏗️ Architecture Overview

Data → Airflow → ML Training → MLflow Tracking → Docker → Vertex AI Deployment  

- Airflow: Orchestrates ML workflows  
- MLflow: Tracks experiments and models  
- Docker: Containerizes ML applications  
- Vertex AI: Model deployment and serving
