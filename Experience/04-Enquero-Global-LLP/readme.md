### Driscoll’s – Cloud API Integration Platform (AWS + GCP)

This project represents a large-scale enterprise implementation focused on building and managing API-driven integrations using MuleSoft, with a multi-cloud setup across AWS and GCP.

The architecture follows an API-led connectivity approach using System APIs, Process APIs, and Experience APIs.

---

## 🏗️ Architecture Overview

Client Applications → Experience API → Process API → System API → Backend Systems  

- Experience APIs: Exposed to frontend/mobile clients  
- Process APIs: Handle business logic and orchestration  
- System APIs: Connect to backend systems (databases, third-party services)
